# blueMSX 用マシン構成ファイル
blueMSX 用の MSX0 マシン構成ファイルです。<br>
blueMSX の `Machines` フォルダに上書きすると、MSX0 のマシン構成が追加されます。

## ROM

ダンプした ROM はそれぞれ次のサブフォルダに格納してください。
