# MSX0 用 Disk イメージ
MSX0 用の Disk イメージです。Turbo Pascal に関係ないものも含まれています。

## ■ DOS2_MSXC.DSK
MSX-C の環境を作るディスクです。

`MSXDOS2.SYS` と `COMMAND2.COM` を組み込んで、このディスクから起動したら、

```
A:\>COPY AUTOINST.BAT H:
A:\>H:
H:\>AUTOINST
```

とする事で、このディスクが MSX-C ver1.2 の開発用ディスクとなります。
